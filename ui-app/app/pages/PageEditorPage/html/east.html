Notes

  • Create a new application
  • Use Preview to see it
  • Use the setup tab to see / edit details
  • When setup details are changed, preview again to see changes
  • Can edit UI elements in Visual Studio Code and just hit refresh on preview screen

Deployment Notes

  • Use the Deploy tab to build the deployment files
  • Then use the open in VS code to get to the terminal
  • In the terminal, you need to log in one time - then you can deploy over and over
  • To log in to bluemix, use the login.bat (sorry mac folks for now)
  • Use login yourusername@us.ibm.com orgname space (see below)
  • You can use npm install (one time)
  • Then npm start as usual
  • Use control - click to open the site at 33462
  • If it looks good, ctrl-c to break out
  • Then use "tocloud" bat file to push out
  • .. or type this in manually (whoa)
  • ibmcloud app push
  • ... and enjoy
Params for login.bat 
param 1 = yourname@us.ibm.com
 (i.e. joefran@us.ibm.com)
param 2 = Org
 (i.e. joefran)
param 3 = Space
 (i.e. dev)
