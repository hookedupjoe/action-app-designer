What is Action Application?
Getting Started

Action Application is a UI Framework than can run on any web server. Based on jQuery and related libraries, the system is designed to make web and mobile application development simple yet powerful.

About the Application Model

  • Easy to learn, implement and maintain
  • Powerful features are available out of the box for all applications and application pages
  • Pages are lazy loaded so the entire site does not load at once
  • Can use Raw HTML, Templating and Controls together to make powerful applications quickly
  • Same UI model works for web pages and mobile apps
  • Easy to extend and maintain -built for enterprise applications and native mobile applications in mind
  • Server agnostic, the same UI can run vitually anywhere

Getting Started
Let's Go!

To get started you can use any of these techniques to create a new application.

Create a Node.js application using the CDN

This version / repository links to the CDN, making it super light weight.
Download the repo, put in a folder, run npm install and npm start to install and run as usual. https://github.ibm.com/joefran/action-app-using-cdn
Note: This will start at port 7072, you should change that in the app.js (in the root) for your application.
Show Video - Creating App From Repo

Copy this code base / repository

Don't want to link to a CDN? Download the CDN and use it as a starting point. You can pull in the library folders anytime to update your application to the latest code.
https://github.com/hookedupjoe/action-app-web
Note: This will start at port 7071, you should change that in the app.js (in the root) for your application.

Use XPages
